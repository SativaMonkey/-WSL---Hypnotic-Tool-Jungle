Результат нічного спринту. Чистий vibecode. Повністю автономне генеративне аудіо-візуальне середовище, запаковане в один файл TUNE.html. Жодних зовнішніх залежностей, фреймворків чи медіа-ассетів. Вся генерація звуку та 3D-графіки відбувається локально на стороні клієнта через нативні Web Audio API та HTML5 Canvas.

Повна оффлайн-функціональність.

Boot Sequence
Єдина точка входу. Запуск ініціює термінальний стартовий екран із ASCII-логотипом. Активація системи (Initialize System) запускає консоль діагностичного завантаження, після чого відбувається автоматичне розгортання інтерфейсу із каскадною появою функціональних панелей.

Generative Engine & Math
Логіка побудована на математичних послідовностях, що робить кожен запуск унікальним.

Алгоритмічне ядро: Мелодійні лінії, вибір пресетів тембру, переходи акордів та ймовірність ритмічних збивок генеруються на основі обробки символів числа Пі (piStr) та ряду Фібоначчі (fib).

Dynamic Seed: При кожній ініціалізації генеруються нові стартові індекси (pIdx, fIdx, phraseCount). Ідентичних сесій не існує.

Гармонічна сітка: Обертання дорійських акордних послідовностей (до 5 голосів). Транспоновані для вивільнення суб-басового діапазону. Мотиви арпеджіатора.

Sound Design & Targets
Вайб: Deep House / Dystopian Ambient.

Low-End: Глибокі синусоїдальні суб-баси із портаменто-ковзанням.

Drums: Насичений аналоговий Kick-click, багатокомпонентний Snare-клап, Roland-стилізовані металеві хай-хети.

Procedural Layers: Генеративна перкусія (liquid drips, organic liquid foley, phonk cowbells, congas, vocal chants).

FX: Ефект стрічкового девайсу (Wow/Flutter LFO) для деградації звуку.

Visual Pipeline (3D Canvas)
Матричні 3D-проєкції: Розрахунок тривимірних координат вершин для 6 режимів (3D DNA Helix, Cylinder Mesh, Particle Wave Ring, Axonometric Step Grid, Terrain Wave). Матричне обертання, проєкційне масштабування та морфінг між станами.

Audio-Reactive: Процедурна анімація зоряного неба та пилу. Швидкість наближення жорстко прив'язана до амплітуди низьких (bassAmt) та середніх частот.

Interactive Control System
Система оптимізована під touch-інтерфейси (passive-listeners, блокування скролу сторінки, нульова затримка відгуку).

XY Pad:

X-Axis: Майстер-фільтр (Lowpass ліворуч / Highpass праворуч). Центр — мертва зона (Bypass).

Y-Axis: Частота LFO-модуляції саб-басу (1Hz - 20Hz).

Soundboard Interrupts: Миттєве накладання ваншотів поверх міксу через виділену шину реверберації/ехо (Orca Call, Sonar Ping, Singing Bowl, Abyssal Swell).

DJ Tape-Stop [SPACE]: Глобальний плавний глайд BPM до значення 10 із приглушенням гучності (ducking). Відпускання відновлює оригінальний темп.

Deployment
Local: Відкрити TUNE.html у браузері.
Vercel: Статичний деплой. Build commands не потрібні. Root Directory: ./
